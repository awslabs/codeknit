Cover these topics in order:

1. Prerequisites: Go 1.26+, a C compiler (CGo needed for tree-sitter)
2. Installation from source:
   ```bash
   git clone https://github.com/awslabs/codeknit.git
   cd codeknit
   make build
   # Binary is at ./bin/codeknit
   ```
3. Add to PATH (bash/zsh/fish examples)
4. Verify: `codeknit --version`
5. First parse — run `codeknit parse ./myproject` and explain what happens (writes .skt files to ./skeleton/)
6. Reading the output — briefly explain the .skt format sections ([symbols], [edges]) with a small example
7. Next steps — link to the parse command guide, graph commands guide, and output format reference.
